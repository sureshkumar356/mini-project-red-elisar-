RED ELISAR OPTION-WISE SCENARIO CHECKLIST

OPTION 1 - FULL WEB VULNERABILITY SCAN
Scenarios to test (at least 5):
1. Baseline scan on http://127.0.0.1:5000 and verify all report files are generated.
2. Exposed resource detection for /.env, /backup, /admin, /api/users.
3. SQL Injection detection and primary MITRE mapping to T1190.
4. Header and CORS weakness detection with relevant ATT&CK mapping.
5. Form discovery plus form-driven SQLi/XSS findings included in final report.
6. Duplicate merge check for same endpoint + same vulnerability type.


OPTION 2 - GENERATE ATTACK SCENARIO (NO URL)
Scenarios to test (at least 5):
1. SQL injection focused scenario narrative.
2. Credential exposure scenario (.env / backup leakage chain).
3. CORS plus missing-security-header scenario.
4. XSS plus session hijack style scenario.
5. Mixed web attack path (initial access -> execution -> data exposure).
6. Validate chain readability and logical progression.


OPTION 3 - VALIDATE SCENARIO ON TARGET URL
Scenarios to test (at least 5):
1. Validate SQLi scenario against /search and /login.
2. Validate exposed-file scenario on /.env and /backup.
3. Validate CORS scenario using target response behavior.
4. Validate auth bypass scenario on login flow.
5. Validate reflected XSS scenario on discovered input endpoints.
6. Confirm probe evidence and recommendations are present in reports.


OPTION 4 - ANALYZE SCENARIO ONLY
Scenarios to test (at least 5):
1. Credential theft to exfiltration scenario analysis.
2. SQLi to database extraction scenario analysis.
3. XSS to credential/session abuse scenario analysis.
4. Open redirect plus phishing-assisted access scenario analysis.
5. Defense-evasion style scenario with weak headers and debug leaks.
6. Confirm technique chain is concise, realistic, and context-mapped.


OPTION 5 - EXIT
Scenarios to test (at least 5):
1. Exit immediately after app launch.
2. Exit after completing Option 1.
3. Exit after completing Option 2.
4. Exit after completing Option 3.
5. Exit after completing Option 4.
6. Confirm no crash and clean CLI termination every time.
